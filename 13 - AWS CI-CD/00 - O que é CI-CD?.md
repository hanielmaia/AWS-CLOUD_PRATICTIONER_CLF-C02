# O que é CI-CD?

O CI/CD (Continuous Integration / Continuous Deployment) é simplesmente uma esteira automatizada que pega o código da máquina do programador e o leva até o servidor final (produção), garantindo que ele seja testado e empacotado no meio do caminho para que nada chegue quebrado no cliente.

- A sequência da Esteira é: CodeCommit - CodeBuild - CodeDeploy
