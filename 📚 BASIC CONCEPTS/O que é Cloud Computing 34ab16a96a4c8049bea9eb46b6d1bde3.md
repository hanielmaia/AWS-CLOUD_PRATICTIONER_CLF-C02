# O que é Cloud Computing?

### O que é Computação em Nuvem?

- ⭐️ É a entrega sob demanda de poder computacional, armazenamento de banco de dados, aplicações e outros recursos de TI por meio da internet, com definição de preço baseada no **pagamento conforme o uso (pay-as-you-go)**.

> 
> 
> - **⭐️ Dica para o Exame:** A nuvem transforma despesas fixas de capital (comprar servidores físicos - CAPEX) em despesas variáveis (pagar apenas pelo que consome - OPEX).
