# Bancos de Dados para Casos de Uso Específicos

Geralmente, na prova do CCP, se a questão descrever um destes cenários específicos, a resposta é direta.

- **Amazon Neptune:** Banco de dados de **grafos** (Graph Database). Usado para trabalhar com dados altamente conectados. *Palavras-chave: Redes sociais, mecanismos de recomendação, detecção de fraudes.*
- **Amazon Timestream:** Banco de dados de **séries temporais**. Escala automaticamente para processar trilhões de eventos por dia. *Palavras-chave: Dados de telemetria de IoT, rastreamento de frotas, logs operacionais ao longo do tempo.*
- **Amazon QLDB (Quantum Ledger Database):** Banco de dados de **livro-razão (ledger)** totalmente gerenciado. Fornece um log de transações transparente, imutável e verificável criptograficamente. *Palavras-chave: Histórico imutável, rastreamento de cadeia de suprimentos, registro financeiro.*
