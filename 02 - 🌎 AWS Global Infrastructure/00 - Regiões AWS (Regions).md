# 00 - Regiões AWS (Regions)

⭐️ Uma Região AWS é uma área geográfica ampla e física no mundo (ex: São Paulo, Norte da Virgínia, Tóquio) onde a AWS hospeda sua infraestrutura. Cada região é completamente independente e isolada das outras para garantir a máxima estabilidade e tolerância a falhas em nível global.

**⭐️ Dica para o Exame: A prova frequentemente pergunta como você escolhe uma Região. Os três fatores principais são: Conformidade legal/governança (os dados não podem sair do país), Latência (ficar o mais perto possível dos seus clientes) e Preço/Disponibilidade de Serviços (os preços variam por região e nem todo serviço está disponível em todas elas).**
