# 💰 4. Otimização de Custos

Aqui o foco é garantir que você **não gaste mais do que precisa**.

Principais práticas:

- **Modelo pay-as-you-go** → pagar só pelo uso
- **Monitoramento de custos** → AWS Cost Explorer, budgets
- **Tipos de cobrança otimizados** → instâncias reservadas, spot
- **Eliminar desperdícios** → recursos ociosos (EC2 parada, volumes não usados)
- **Dimensionamento correto (rightsizing)**

💡 Em resumo: custo na nuvem não é fixo — você controla totalmente quanto gasta.
