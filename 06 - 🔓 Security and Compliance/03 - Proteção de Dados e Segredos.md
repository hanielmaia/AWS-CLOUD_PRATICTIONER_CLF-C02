# Proteção de Dados e Segredos

- ⭐️ **AWS KMS (Key Management Service):** Serviço gerenciado para criar e controlar as **chaves de criptografia** usadas para criptografar seus dados. Fica integrado a quase todos os serviços da AWS.
- ⭐️ **AWS CloudHSM:** Módulo de segurança de hardware dedicado em nuvem. Ao contrário do KMS (que é compartilhado), o CloudHSM te dá controle total sobre um hardware de criptografia real, muito exigido para conformidade governamental rígida.
- ⭐️ **AWS Secrets Manager:** Protege, armazena e, o mais importante, **rotaciona automaticamente** credenciais de banco de dados, chaves de API e outras senhas.
- ⭐️ **AWS Certificate Manager (ACM):** Provisiona, gerencia e implanta certificados SSL/TLS públicos e privados de forma fácil (para garantir o "HTTPS" no seu site).
