# Serviços Prontos (Aplicações de IA)

Estes são serviços onde a inteligência já está treinada pela AWS. Você apenas envia os dados e recebe o resultado.

⭐️ **Amazon Rekognition:** É um serviço de análise de imagem e vídeo que usa ML para identificar objetos, pessoas, textos, cenas e atividades. Ele também é usado para detecção de conteúdo inadequado e reconhecimento facial seguro.

⭐️ **Amazon Comprehend:** É um serviço de Processamento de Linguagem Natural (NLP) que usa ML para encontrar percepções (insights) e relacionamentos em textos. Ele entende o tom emocional do que está escrito.

⭐️ **Amazon Lex:** É um serviço para criar interfaces de conversação (texto e voz) para qualquer aplicação. É literalmente o mesmo motor de inteligência artificial que está por trás da Amazon Alexa.

⭐️ **Amazon Polly:** É um serviço que transforma texto em fala realista (Text-to-Speech). Ele permite criar aplicações que falam, construindo produtos totalmente novos baseados em voz.

⭐️ **Amazon Transcribe:** Faz exatamente o oposto do Polly: ele converte fala em texto (Speech-to-Text) de forma automática usando reconhecimento de fala.

⭐️ **Amazon Translate:** É um serviço de tradução automática neural de alta qualidade e rapidez, que traduz textos entre dezenas de idiomas diferentes de forma muito natural.

⭐️ **Amazon Kendra:** É um serviço de pesquisa corporativa altamente inteligente. Ele permite que seus funcionários façam perguntas em linguagem natural e encontrem respostas precisas espalhadas pelos documentos, manuais e intranets da empresa.

⭐️ **O Amazon Textract:** Serviço da AWS que usa machine learning para extrair automaticamente texto, formulários e tabelas de documentos digitalizados, como PDFs e imagens. Diferente de um OCR básico, ele entende a estrutura do documento, conseguindo identificar campos de formulários (como nome, data, valores) e organizar informações em tabelas de forma estruturada. Isso permite automatizar processos manuais, como leitura de notas fiscais, contratos ou documentos administrativos, reduzindo erros e economizando tempo no processamento de grandes volumes de dados.

⭐️ **Amazon Personalize:** O Amazon Personalize é um serviço de machine learning totalmente gerenciado que os clientes podem usar para gerar recomendações personalizadas para seus usuários. Ele também pode gerar segmentos de usuários com base na afinidade dos usuários por determinados itens ou metadados de itens. O Amazon Personalize usa os dados dos clientes para treinar e implantar modelos de recomendação personalizados que podem ser integrados em suas aplicações.
