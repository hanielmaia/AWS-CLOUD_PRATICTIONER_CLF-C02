# ☁️ Sobre o Exame — AWS Certified Cloud Practitioner (CLF-C02)

---

## 📋 Visão Geral

O **AWS Certified Cloud Practitioner (CLF-C02)** é a certificação de nível **foundational** da Amazon Web Services. Ela valida uma compreensão geral da nuvem AWS — seus serviços, conceitos, segurança, arquitetura, preços e suporte — sendo ideal para quem está **iniciando na nuvem** ou migrando de outra área para TI/Cloud.

Não é exigida experiência técnica prévia com AWS para realizar o exame, mas familiaridade com conceitos básicos de TI é recomendada.

---

## 🕐 Formato do Exame

| Característica | Detalhe |
|---|---|
| **Código** | CLF-C02 |
| **Nível** | Foundational |
| **Duração** | 90 minutos |
| **Número de questões** | 65 questões |
| **Questões pontuadas** | 50 questões |
| **Questões não pontuadas** | 15 questões (sem identificação) |
| **Formato das questões** | Múltipla escolha e múltipla resposta |
| **Pontuação mínima para aprovação** | 700 / 1000 |
| **Idioma disponível** | Inglês, Português (BR), e outros |
| **Modalidade** | Presencial (centro de testes) ou Online Proctored |
| **Validade da certificação** | 3 anos |
| **Custo** | USD 100 |

> ⚠️ As 15 questões não pontuadas são usadas pela AWS para avaliar futuras versões do exame. Você não saberá quais são.

## 📚 Domínios Cobrados

O exame CLF-C02 é dividido em **4 domínios**:

| # | Domínio | Peso |
|---|---|---|
| 1 | Cloud Concepts *(Conceitos de Nuvem)* | 24% |
| 2 | Security and Compliance *(Segurança e Conformidade)* | 30% |
| 3 | Cloud Technology and Services *(Tecnologia e Serviços)* | 34% |
| 4 | Billing, Pricing and Support *(Faturamento e Suporte)* | 12% |

---

## 💡 Dicas Gerais

- Leia todas as alternativas antes de marcar — a AWS costuma usar distratores parecidos com a resposta correta.
- Em questões de múltipla resposta, o enunciado indicará **quantas respostas** selecionar (ex: *"Escolha 2"*).
- Gerencie seu tempo: com 90 min para 65 questões, você tem **~1min 23seg por questão**.
- Não existe penalidade por resposta errada — **nunca deixe em branco**.
