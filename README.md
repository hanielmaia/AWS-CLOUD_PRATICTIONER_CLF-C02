# AWS-CLOUD_PRATICTIONER_CLF-C02
Este repo é dedicado aos meus estudos para a certificação AWS Certified Cloud Practitioner, com resumos organizados dos principais conceitos da Amazon Web Services. Este repositório serve tanto como base de conhecimento pessoal quanto como registro da minha jornada rumo à certificação. Sinta-se à vontade para contribuir, ou utilizar este material nos seus próprios estudos! 

## O conteúdo inclui tópicos como:

- Fundamentos de cloud computing
- Modelos de serviço (IaaS, PaaS, SaaS)
- Principais serviços da AWS (EC2, S3, RDS, Lambda, etc.)
- Segurança e conformidade
- Modelos de precificação e billing
