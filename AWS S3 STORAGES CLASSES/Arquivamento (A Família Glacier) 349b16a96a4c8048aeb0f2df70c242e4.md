# Arquivamento (A Família Glacier)

⭐️ **S3 Glacier Instant Retrieval:** Classe de arquivamento que oferece o armazenamento de custo mais baixo para dados de vida longa que raramente são acessados (uma vez por trimestre), mas que precisam de recuperação em **milissegundos** quando solicitados.

⭐️ **S3 Glacier Flexible Retrieval (antigo S3 Glacier):** Para arquivos que podem esperar de **minutos a horas (até 12h)** para serem recuperados. É significativamente mais barato e oferece opções flexíveis de velocidade de recuperação e custo (Expedited, Standard, Bulk).

⭐️ **S3 Glacier Deep Archive:** A classe de armazenamento absolutamente **mais barata** de toda a AWS. O tempo de recuperação padrão varia de **12 a 48 horas**.
