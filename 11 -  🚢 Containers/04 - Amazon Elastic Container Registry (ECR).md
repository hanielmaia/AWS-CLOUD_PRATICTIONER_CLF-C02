# Amazon Elastic Container Registry (ECR)

⭐️ O Amazon ECR é um registro de containers Docker totalmente gerenciado que facilita para os desenvolvedores o armazenamento, o gerenciamento e a implantação de imagens de containers. Ele é integrado ao ECS e ao EKS para facilitar o fluxo de trabalho de desenvolvimento para produção.

⭐️ Dica para o Exame: Pense no ECR como o "Docker Hub da AWS". Se a questão mencionar "armazenar imagens de container de forma segura e privada", a resposta é o ECR.
