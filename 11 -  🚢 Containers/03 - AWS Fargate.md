# AWS Fargate

⭐️ O AWS Fargate é um mecanismo de computação sem servidor (serverless) para containers que funciona tanto com o ECS quanto com o EKS. Com o Fargate, você não precisa mais gerenciar ou provisionar servidores EC2; você apenas define os requisitos de CPU e memória e a AWS cuida do resto.

⭐️ Dica para o Exame: As palavras-chave são "Serverless para containers" e "sem gerenciar instâncias EC2". Se o desenvolvedor quer rodar containers sem se preocupar com a manutenção do Sistema Operacional das máquinas virtuais, escolha o Fargate.
