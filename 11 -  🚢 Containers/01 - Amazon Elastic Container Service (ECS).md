# Amazon Elastic Container Service (ECS)

⭐️ O Amazon ECS é um serviço de orquestração de containers totalmente gerenciado, altamente escalável e de alto desempenho. Ele permite que você execute e gerencie facilmente aplicações em containers Docker na AWS, integrando-se nativamente com outros serviços como o Elastic Load Balancing e o IAM.

⭐️ Dica para o Exame: É o orquestrador proprietário da AWS. Se a questão falar em "simplicidade de integração" com a AWS para rodar containers Docker, o ECS é a resposta.
