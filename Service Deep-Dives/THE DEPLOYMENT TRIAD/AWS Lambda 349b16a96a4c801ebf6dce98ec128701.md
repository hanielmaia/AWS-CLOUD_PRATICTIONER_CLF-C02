# AWS Lambda

- ⭐️ O AWS Lambda é um serviço de computação sem servidor (*serverless*) que permite executar trechos de código em resposta a eventos (como o upload de um arquivo no S3 ou uma requisição web) sem precisar provisionar ou gerenciar nenhum servidor, cobrando de você apenas pelos milissegundos exatos em que o código esteve rodando.
- **⭐️ Dica para o Exame:** As palavras-chave mágicas são "**serverless**", "**orientado a eventos**", tempo de execução de "**milissegundos**" e cenários onde o desenvolvedor quer executar funções curtas de backend sem tocar em infraestrutura.
