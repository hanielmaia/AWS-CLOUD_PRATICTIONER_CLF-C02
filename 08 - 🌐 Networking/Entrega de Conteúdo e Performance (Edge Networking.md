# Entrega de Conteúdo e Performance (Edge Networking)

- **Amazon Route 53:** Serviço de DNS (Domain Name System) altamente disponível. Ele roteia usuários para aplicações e verifica a saúde dos recursos.
- **Amazon CloudFront:** Rede de Entrega de Conteúdo (CDN) que acelera a distribuição de dados, vídeos e APIs para usuários globalmente através de *Edge Locations*.
- **AWS Global Accelerator:** Melhora a disponibilidade e performance das suas aplicações usando a rede global da AWS para otimizar o caminho do tráfego.
