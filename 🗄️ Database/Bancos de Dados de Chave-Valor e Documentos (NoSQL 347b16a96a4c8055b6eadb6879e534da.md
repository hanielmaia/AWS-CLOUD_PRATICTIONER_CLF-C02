# Bancos de Dados de Chave-Valor e Documentos (NoSQL)

Ideais para dados não estruturados ou semi-estruturados, esquemas flexíveis e escala massiva (milissegundos de latência).

- ⭐️ **Amazon DynamoDB:** Banco de dados de chave-valor e documentos sem servidor (serverless). Oferece latência de um dígito de milissegundo em qualquer escala. Muito usado para aplicações web de alto tráfego, jogos e IoT.
- **Amazon DocumentDB:** Serviço de banco de dados de documentos gerenciado, rápido, escalável e altamente disponível, construído com compatibilidade nativa para cargas de trabalho do MongoDB.
- **Amazon Keyspaces (for Apache Cassandra):** Serviço de banco de dados compatível com Apache Cassandra, altamente disponível e gerenciado (modelo wide-column).
