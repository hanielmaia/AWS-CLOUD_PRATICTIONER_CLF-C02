# Bancos de Dados Relacionais (SQL / OLTP)

Ideais para dados altamente estruturados, esquemas rígidos e transações complexas (como sistemas de ERP, CRM e e-commerce tradicional).

- ⭐️ **Amazon RDS (Relational Database Service):** Serviço gerenciado que facilita a configuração, operação e escalabilidade de bancos de dados relacionais. Suporta motores comerciais e de código aberto (MySQL, PostgreSQL, MariaDB, Oracle, SQL Server).
- ⭐️ **Amazon Aurora:** Um banco de dados relacional construído pela própria AWS para a nuvem. É compatível com MySQL e PostgreSQL, mas oferece até 5x o desempenho do MySQL padrão com o custo de bancos de dados de código aberto. É altamente disponível e tolerante a falhas.
